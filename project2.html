Overview

This project focuses on optimizing dairy farming practices by predicting the most suitable storage conditions for dairy products. Machine learning models analyze key farm and product attributes to provide actionable recommendations for dairy managers. The solution is integrated with a Flask web interface for real-time predictions.

Technology Stack

  • Python (Pandas, Label Encoding, Joblib)
  • ML Models: KNN, Decision Tree Classifier, Random Forest Classifier
  • Web App: Flask for real-time predictions
  • GitHub Repository for code and models
