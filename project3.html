Concrete Strength Prediction

Professional Project | Feb 2025 – Mar 2025

Overview

This project applies advanced regression techniques to predict the compressive strength of concrete based on mix attributes. Models used include Linear Regression, KNN Regressor, and Support Vector Regressor (SVR). The goal is to provide accurate, data-driven predictions for civil engineering applications.

Technology Stack

  • Python (Pandas, NumPy, Joblib, Label Encoding)
  • ML Models: Linear Regression, KNN Regressor, SVR
  • Web App: Flask (for building real-time prediction interface)
  • Visualization & Analysis: Matplotlib, Seaborn
  • Version Control: GitHub repository with code and trained models

Workflow

  1. Data Collection & Preprocessing: Gathered concrete mix dataset, handled missing values, normalized features, and applied label encoding.
  2. Exploratory Data Analysis (EDA): Visualized relationships, correlations, and outliers.
  3. Feature Selection: Identified key features affecting concrete strength for optimal model performance.
  4. Model Training & Evaluation: Trained Linear Regression, KNN Regressor, and SVR models; evaluated using MSE, RMSE, and R² Score.
  5. Prediction & Visualization: Predicted concrete strength and visualized predicted vs actual values to assess accuracy.

Model Performance & Outcomes

  • Linear Regression: MSE 95.97%, RMSE 9.8, R² 63.0%, Predicted Strength 52.43 MPa
  • KNN Regressor: MSE 68.07%, RMSE 8.25, R² 74.0%, Predicted Strength 70.94 MPa
  • SVR: MSE 66.28%, RMSE 8.14, R² 74.0%, Predicted Strength 52.36 MPa
