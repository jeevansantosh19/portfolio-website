  2. Exploratory Data Analysis (EDA): Performed statistical analysis and visualizations to understand relationships between features and target variable. Identified key patterns, correlations, and potential outliers.
  3. Feature Selection: Selected the most influential features affecting concrete strength using correlation analysis and domain knowledge, ensuring optimal model performance.
  4. Model Training and Evaluation: Trained multiple regression models (Linear Regression, KNN Regressor, SVR) on the dataset. Evaluated performance using metrics such as MSE, RMSE, and RÂ² Score to compare model effectiveness.
  5. Prediction and Visualization: Used the trained models to predict concrete strength on test data. Visualized predicted vs actual values to assess model accuracy and identify areas for improvement.
