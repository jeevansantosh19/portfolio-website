BlinkIT Dashboard

Professional Project | Feb 2025 – Mar 2025

Overview

The BlinkIT Dashboard project focuses on analyzing sales and product data to extract meaningful insights. It highlights key retail metrics such as sales performance, item distribution, and customer ratings. With Excel preprocessing and Power BI integration, the project ensures efficient data cleaning, DAX-driven measures, and dynamic visualization. The outcome is a business-ready dashboard that supports smarter decisions in retail operations and inventory management.

Key Performance Indicators (KPIs)

  • Total Sales
  • Average Sales
  • Number of Items
  • Average Rating

Goal

Deliver actionable insights to support decision-making in retail operations and inventory planning.

Workflow

  1. Collected raw BlinkIT dataset for analysis.
  2. Cleaned and preprocessed data using Power BI Query Editor.
  3. Created DAX measures for sales, items, and ratings.
  4. Built interactive visualizations in Power BI.
  5. Generated insights to guide retail strategy and inventory management.

Technology Stack

  • Excel – Data exploration and formatting
  • Power BI Query Editor – Data cleaning, handling nulls, creating calculated columns
  • Power BI – DAX measures and dynamic dashboards
