BlinkIT Dashboard

Overview

The BlinkIT Dashboard project focuses on analyzing sales and product data to extract meaningful insights. It emphasizes key retail metrics such as sales performance, item distribution, and customer ratings. By integrating Power BI with Excel preprocessing, the project ensures efficient data cleaning, DAX-driven measures, and dynamic visualization. The outcome is a business-ready dashboard that supports smarter decisions in retail operations and inventory management.

Key Performance Indicators (KPIs)

  • Total Sales
  • Average Sales
  • Number of Items
  • Average Rating

Goal

Deliver business-ready insights to support informed decision-making in retail operations and inventory planning.

Workflow

  1. Data Collection from BlinkIT dataset
  2. Data Cleaning & Preprocessing using Power BI Query Editor
  3. Creating DAX Measures for metrics like Sales, Items, and Ratings
  4. Building Interactive Visualizations in Power BI
  5. Generating Insights to support retail and inventory planning

Technologies Used

  • Excel – Data exploration and formatting
  • Power BI Query Editor – Initial data cleaning, handling nulls, and replacing placeholders with real values
  • Power BI – DAX measures and dynamic visualizations
