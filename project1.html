Animal Health Monitoring & Disease Detection

Final Year Major Project | IV Year II Semester | 2023–2024

Overview

This project focuses on developing a machine learning–based system for monitoring animal health and detecting early signs of diseases. The Random Forest Classifier was implemented to analyze health-related datasets and predict potential diseases with high accuracy. The solution eliminates the dependency on IoT devices and emphasizes data-driven decision making.

Technology Stack

  • Data Source: CSV File
  • Tools: Python (Jupyter Notebook), Flask
  • Frontend: HTML with Embedded CSS
  • Machine Learning Model: Random Forest Classifier
  • Evaluation Metrics: Accuracy, Precision, Recall, Probability, Predicted Values

Project Outcomes

  • Achieved Accuracy: ~79%
  • Probability Score: 88%
  • Predicted Disease: Pneumonia

Workflow

  1. Data collection and preprocessing using Pandas.
  2. Exploratory Data Analysis and feature engineering.
  3. Training and evaluation of the Random Forest Classifier.
  4. Prediction and probability calculation for new samples.
  5. Visualization of results and reporting.
