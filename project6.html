boAT Electronics Dashboard

Professional Project | Jan 2025 – Feb 2025

Overview

This project showcases a Power BI Dashboard for boAT Electronics, providing insights into sales, profitability, and operational efficiency. It enables stakeholders to track KPIs, monitor performance, and optimize delivery dynamics through interactive data visualizations.

Technology Stack

  • Excel – Cleaned raw dataset, removed null values
  • Power BI Query Editor – Data transformation, custom columns, segmentation tables
  • Power BI – DAX measures, data modeling, responsive visual dashboard

Workflow

  1. Imported and cleaned sales data using Excel.
  2. Transformed data in Power BI Query Editor; created calculated columns (e.g., Avg Delivery Time).
  3. Defined DAX measures for total sales, profit, quantity, and delivery metrics.
  4. Designed an interactive dashboard with filters, slicers, and advanced visuals in Power BI.

Key Performance Indicators (KPIs)

  • Total Sales
  • Total Profit
  • Total Quantity Sold
  • Average Delivery Time
