boAT Electronics Dashboard

Overview

This project showcases a Power BI dashboard for boAT Electronics, offering insights into sales, profitability, and operational efficiency. It helps stakeholders monitor performance and optimize delivery dynamics through interactive visual analytics and KPI tracking.

Technology Stack
  • Excel – Cleaned raw dataset, removed nulls
  • Power BI Query Editor – Data transformation, custom columns (e.g., Avg Delivery Time), segmentation tables
  • Power BI – DAX measures, data modeling, and responsive visual dashboard
Workflow
  1. Imported and cleaned sales data using Excel.
  2. Used Power BI Query Editor to transform data and create new calculated columns.
  3. Defined DAX measures for total sales, profit, quantity, and avg delivery time.
  4. Designed interactive dashboard with slicers and visuals in Power BI.
Key Performance Indicators (KPIs)
  • Total Sales
  • Total Profit
  • Total Quantity Sold
  • Average Delivery Time
