Overview

The Adidas Sales Dashboard project focuses on providing data-driven insights to monitor and optimize sales performance. The dashboard consolidates monthly sales trends, regional and product-level performance, and KPI metrics to help stakeholders make strategic business decisions. Interactive visualizations allow drill-down analysis by region, product category, and invoice date.

Technology Stack

  • Data Source: Excel
  • Data Cleaning: Query Editor (Handled nulls, ensured data quality)
  • Visualization & Dashboard: Power BI
  • GitHub Repository for project files and dataset samples
