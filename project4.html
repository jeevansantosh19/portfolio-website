Revolutionizing Dairy Practices Using Machine Learning

Using AI to enhance dairy product classification for efficiency.

Project Overview

This project explores machine learning for classifying dairy products into categories like Refrigerated and Tetra Pack. By leveraging supervised learning, we improve efficiency and decision-making in dairy production and logistics.

  • Algorithms Used: K-Nearest Neighbors (KNN), Decision Tree Classifier, Random Forest Classifier
  • Tools: Python, Scikit-Learn, Pandas, NumPy, Matplotlib

Performance Summary

We implemented three classification models to categorize dairy products based on their characteristics. K-Nearest Neighbors (KNN) achieved an accuracy of 67.17%, indicating a basic level of classification performance. The Decision Tree Classifier significantly improved classification with an accuracy of 80.92%, demonstrating better decision boundaries. The Random Forest Classifier outperformed both models, achieving the highest accuracy of 85.43%, highlighting its robustness in handling complex data relationships. These results showcase the effectiveness of machine learning in optimizing dairy product classification for improved storage and logistics.
