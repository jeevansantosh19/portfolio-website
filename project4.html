Overview

The Adidas Sales Dashboard project consolidates monthly sales trends, regional breakdowns, and product category insights into a Power BI dashboard. Stakeholders can quickly track performance via key sales metrics and drill down by region or date.

Technology Stack

  • Data Source: Excel
  • Query Editor: Handled missing values & ensured data quality
  • Visualization & Dashboard: Power BI
  • GitHub repository for dashboard files

Workflow

  1. Imported and cleaned sales data using Excel and Power BI Query Editor.
  2. Created data model linking regions, dates, and product categories.
  3. Designed interactive dashboard with monthly trends, regional maps, and product insights.
  4. Enabled drill-down by region and invoice date for detailed analysis.
  5. Validated KPIs and published dashboard for stakeholder use.

Key Performance Indicators (KPIs)

  • Total Sales
  • Operating Profit
  • Units Sold
  • Average Price per Unit
  • Average Operating Margin
