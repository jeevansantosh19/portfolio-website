Kevin Cookie Company Dashboard

Professional Project | 2025

Overview

The Kevin Cookie Company Dashboard provides an in-depth analysis of global cookie sales, revenue, and profit metrics. Using Excel, Power Query Editor, and Power BI, stakeholders can gain insights into performance trends, operational costs, and profitability.

Key Performance Indicators (KPIs)

  • Total Units Sold
  • Total Profit
  • Total Revenue
  • Average Revenue per Unit
  • Total Cost
  • Profit %

Workflow

  1. Loaded and cleaned sales data in Excel.
  2. Performed transformations and quality checks in Power Query Editor.
  3. Added calculated columns and DAX measures in Power BI Desktop for advanced analysis.
  4. Designed interactive dashboards with KPIs and visuals for insights.

Technology Stack

  • Excel
  • Power Query Editor
  • Power BI Desktop
