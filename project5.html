PayTM User Retention & Revenue Dashboard

Overview

This project focuses on analyzing user retention and revenue patterns for PayTM. Using Power BI, the dashboard provides insights into user acquisition, retention, and churn while highlighting revenue distribution across services. The analysis enables data-driven strategies to enhance user engagement and profitability.

Technology Stack
  • Excel – Initial data source
  • Power BI Query Editor – Data cleaning, transformations, measure creation, segmentation
  • Power BI – Data modeling, interactive dashboard design, and visualizations
Workflow
  1. Loaded and cleaned dataset in Power BI Query Editor.
  2. Performed transformations: handled nulls, added measures, and created new columns.
  3. Built a data model to integrate user and transaction datasets.
  4. Designed Power BI dashboards with interactive filters and KPIs.
  5. Analyzed insights on revenue, user retention, churn, and city-level activity.
Key Performance Indicators (KPIs)
  • Total Revenue
  • Total Transaction Value
  • New Users
  • Retained Users
  • Churned Users
  • Number of Cities
