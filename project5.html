Locomotive Type Classification Using Deep Convolution Neural Networks

Deep Learning Approach for Image-Based Train Classification

Project Overview

This project utilizes Deep Convolutional Neural Networks (CNNs) to classify locomotive types from images. The model is trained on a diverse dataset of locomotive images to enhance automated railway system management.

  • Model Used: Deep Convolutional Neural Network (CNN)
  • Libraries: TensorFlow, Keras, OpenCV, NumPy
  • Dataset: Locomotive images categorized by type
