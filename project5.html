Locomotive Type Classification From Images Using Deep Convolution Neural Networks

Deep Learning Approach for Image-Based Train Classification

Project Overview

This project utilizes Deep Convolutional Neural Networks (CNNs) to classify locomotive types from images. The model is trained on a diverse dataset of locomotive images to enhance automated railway system management.

  • Model Used: Deep Convolutional Neural Network (CNN)
  • Libraries: TensorFlow, Keras, OpenCV, NumPy
  • Dataset: Locomotive images categorized by type
  • Accuracy Achieved: 79%

Performance Summary

The model's performance was evaluated during different stages of training and testing, providing valuable insights into its ability to classify locomotive types effectively. Below is a breakdown of the results:

  • Training Accuracy: 71% – This indicates the model's performance on the training dataset. The value suggests that the model has learned the features of the training data but may still have some room for improvement.
  • Validation Accuracy: 80% – This reflects the model's performance on unseen validation data. A higher validation accuracy compared to training accuracy indicates that the model is generalizing well to new data and is not overfitting.
  • Testing Accuracy: 79% – The testing accuracy demonstrates the model's final performance on the test dataset, which represents real-world data. A high testing accuracy confirms that the model is reliable for locomotive classification tasks.

These results highlight the model's capability to differentiate between locomotive types effectively. While the performance is satisfactory, there is potential for further optimization by fine-tuning the hyperparameters, increasing the dataset size, or employing data augmentation techniques.
