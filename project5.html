PayTM User Retention & Revenue Dashboard

Professional Project | 2025

Overview

The PayTM User Retention & Revenue Dashboard project focuses on analyzing user retention and revenue patterns. Using Power BI, it provides insights into user acquisition, retention, churn, and revenue distribution across services. This enables data-driven strategies to boost engagement and profitability.

Technology Stack

  • Excel – Initial data source
  • Power BI Query Editor – Data cleaning, transformations, measure creation, segmentation
  • Power BI – Data modeling, interactive dashboards, and visualizations

Workflow

  1. Loaded and cleaned dataset in Power BI Query Editor.
  2. Performed transformations: handled nulls, created measures, added calculated columns.
  3. Built a data model integrating user and transaction datasets.
  4. Designed dashboards with interactive filters and KPIs.
  5. Analyzed insights on revenue, retention, churn, and city-level activity.

Key Performance Indicators (KPIs)

  • Total Revenue
  • Total Transaction Value
  • New Users
  • Retained Users
  • Churned Users
  • Number of Cities
