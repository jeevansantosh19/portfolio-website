Coca-Cola Dashboard

Professional Project | 2025

Overview

The Coca-Cola Dashboard project provides a comprehensive analytical Power BI dashboard for global sales, profitability, order trends, and customer satisfaction metrics. The interactive visuals and KPI-driven layout allow stakeholders to make efficient, data-driven decisions.

Technology Stack

  • Excel – Data cleaning and optimization
  • Power Query Editor – Data transformation and standardization
  • Power BI Desktop – Dashboard design, KPI visuals, and interactivity

Workflow

  1. Loaded and cleaned dataset using Excel and Power Query Editor.
  2. Performed transformations, cleaned null values, and standardized formats.
  3. Created measures and calculated columns in Power BI Desktop.
  4. Designed interactive dashboard with KPI cards, slicers, and visuals.
  5. Optimized layout for usability and performance.

Key Performance Indicators (KPIs)

  • Total Sales
  • Unit Price
  • Operational Profit
  • Items Sold
  • Order Count
  • Average Delivery Time
  • Average Customer Rating
