Coca-Cola – The Global Beverage Giant

Overview

This Power BI project delivers a comprehensive analytical dashboard for Coca-Cola’s global metrics, showcasing insights into sales, profitability, order trends, and customer satisfaction. With intuitive visuals and KPI-driven layout, the dashboard enables efficient decision-making across business operations.

Key Performance Indicators (KPIs)

  • Total Sales
  • Unit Price
  • Operational Profit
  • Items Sold
  • Order Count
  • Average Delivery Time
  • Average Customer Rating

Workflow

  • Excel: Initial cleaning of data and optimization of columns.
  • Power Query Editor: Transformed data, cleaned nulls, and standardized formats.
  • Power BI Desktop:
    • Developed custom measures and calculated columns.
    • Designed interactive visuals, KPI cards, and slicers.
    • Applied layout optimizations for usability and performance.

Technology Stack

  • Excel
  • Power Query Editor
  • Power BI Desktop
